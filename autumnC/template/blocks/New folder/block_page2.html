Kiss Portal 0.0.7

KisMod Portal 0.0.7 is a front-end for the new phpBB3, currently available at revision: Beta 5. The current version of KisMod Portal includes, several mod such as Global Images (each theme can have its own images/icons/smilies) this makes for better blending of images and produce a nicer finished product, user Country Flag, etc., See the site for more information... The term "KisBB3" means "Keep it simple" and "integrated modifications" since the added modifications are customized to be compatible with each other.

This version of KisMod is more powerful and feature-packed with the integration of Kiss Portal which is a versatile and powerful homepage frontend for the phpBB3 forum. The version of Kiss Portal installed with KisMod utilizes the forum-wide integration of the blocks so that all pages can display the portal blocks as desired, multi-page feature for additional portal pages like this, utilising phpBB3's caching system for faster performance and lesser resources needed.
